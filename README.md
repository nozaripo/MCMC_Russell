# MSM_Modeling_and_MCMC
Workflow for performing MCMC analyses on OpenSim musculoskeletal models

DRAM_MCMC_Matlab: 
This is the Matlab implementation of the DRAM MCMC algorithm used with this project. It was obtained from this github: https://github.com/mjlaine/mcmcstat and documented further here: https://mjlaine.github.io/mcmcstat/#orgcdeadeb.
